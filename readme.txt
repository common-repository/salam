A plugin for sending SMS otp for login and register to  Woocommerce Sites

== Description ==

By Using this plugin admin and buyer can get otp via sms using SMS Alert.

you can set your dedicated sms gateway(kavenegar.ir) in dashboard panel & let visitors to signup or login to panel.
after  active this plugin on worpdress dashboard just set the api key in setting panel 
and in front-end use check the options for adding mobile login to defualt woocommrce login form


= Salam - (Key Features) =

> + OTP register & login 
> + Combined login & register &  Reset password
> + Login with OTP
> + Supports wordpress multisite
> + Supports Woocommerce
> + Use Every Where you want just with adding  class name :  "logmein"
